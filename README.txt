MAGCENT FAITH BLOG - GITHUB READY

Upload these items to the TOP LEVEL of your GitHub repository:
- index.html
- faith-blog/
- admin/
- content/
- uploads/
- _redirects

After GitHub upload:
1. Go to Netlify
2. Add new site -> Import from GitHub
3. Choose this repo
4. Leave build command empty
5. Leave publish directory empty
6. Deploy

Then in Netlify:
1. Enable Identity
2. Invite your email
3. Enable Git Gateway

Then test:
